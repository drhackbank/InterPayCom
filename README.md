# InterPayCom — GitHub Pages Frontend

This is the **frontend** (GitHub Pages).
The **backend APIs** are in the separate `interpaycom-api` repo (Vercel).

## Setup in 1 step

Open `script.js` and replace:
```js
const API_URL = 'https://YOUR_VERCEL_API_URL_HERE';
```
with your actual Vercel API URL, e.g.:
```js
const API_URL = 'https://interpaycom-api.vercel.app';
```

## Deploy

Push to GitHub → Settings → Pages → main branch → Save
